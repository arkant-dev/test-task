# Про проект

1. У проекті 2 папки **frontend** та **server** у кожного є свій рідмі з коротким описом логіки, я не описував кожний файл.
2. Для запуску frontend відкрийте 1ший термінал

```bash
cd frontend && npm i
```
```bash
npm run start
```

3. Для запуску server відкрийте 2гий термінал

```bash
cd server && npm i
```
```bash
npm run dev
```

4. Підключення до вебсокета відбуться коли ви відкриєте мапу.
5. Щоб знову знегенерувати об'єкти після їх деактивації та виделення (через 5 хвилин) просто перезапустіть сервер і перезавантажте фронтенд сторінку.
6. Елементи зникатимуть згідно ТЗ після 5 хв в статусі disabled.
