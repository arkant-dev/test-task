# Frontend

Фронтенд застосунок побудований на `React`, `TypeScript`, `MobX`, `Material UI` та `Leaflet` для відображення об'єктів із websocket-сервера на мапі.

## Запуск

```bash
npm start
```

Застосунок відкривається за адресою [http://localhost:3000](http://localhost:3000).

## Логіка фронтенду

- `App.tsx` - керує переходом між welcome-екраном і fullscreen-мапою та ініціалізує websocket-підключення.

- `WelcomeScreen.tsx` - показує стартовий екран із коротким описом застосунку.
- `MapScreen.tsx` - рендерить актуальні об'єкти зі стора поверх мапи Leaflet.

- `useTrackedTriangleMarker.ts` - обчислює поворот трикутного маркера, його кольори та текстовий підпис залежно від статусу й напрямку.
- `TrackedObjectMarker.tsx` - малює кожен об'єкт у вигляді витягнутого трикутника, який повертається в напрямку руху.

- `MapOverlayCard.tsx` - відображає інформаційну панель над мапою про кількість і стан об'єктів.

- `AppStore.ts` - відповідає за websocket-з'єднання, обробку повідомлень сервера та оновлення стану підключення.
- `MapStore.ts` - зберігає об'єкти з сервера, готує останні координати для рендеру та рахує активні й втрачені об'єкти.


## Відображення статусів

- `active` об'єкти показуються кольоровими трикутниками з напрямком руху в підписі.
- `disabled` об'єкти показуються сірими трикутниками з підписом `Втрачено зв'язок`.
